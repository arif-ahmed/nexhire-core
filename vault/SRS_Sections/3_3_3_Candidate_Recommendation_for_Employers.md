# 3.3.3. Candidate Recommendation for Employers

*(SRS page 19)*

3.3.3. Candidate Recommendation for Employers
ID Requirement
FR-90. The system SHALL provide employers with candidate recommendations for
their job postings.
FR-91. The system SHALL rank candidates based on match quality, highlighting the
strengths and potential gaps of each candidate.
FR-92. The system SHALL allow employers to set minimum qualification thresholds
for automatic candidate filtering.
FR-93. The system SHALL allow employers to search the candidate database using
advanced filtering options.
FR-94. The system SHALL respect candidate privacy settings when making
recommendations to employers.
FR-95. The system SHALL provide insights on candidate availability, salary
expectations, and potential fit.
FR-96. The system SHALL allow employers to save promising candidates to talent
pools for future opportunities (add to shortlist).
