# 5.8.1. Cultural Considerations

*(SRS page 35)*

5.8.1. Cultural Considerations
ID Requirement
NFR-155. The system SHALL respect cultural norms and sensitivities in the Palestinian
context.
NFR-156. The system SHALL use appropriate terminology and language for the local
context.
NFR-157. The system SHALL support local date and time formats, including Hijri
calendar references where appropriate.
NFR-158. The system SHALL consider gender sensitivities in user interfaces and
communications.
