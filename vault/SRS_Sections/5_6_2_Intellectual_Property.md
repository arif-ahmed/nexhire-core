# 5.6.2. Intellectual Property

*(SRS page 33)*

5.6.2. Intellectual Property
ID Requirement
NFR-123. The system SHALL respect intellectual property rights in all content and
functionality.
NFR-124. The system SHALL properly license all third-party components and libraries.
NFR-125. The system SHALL provide appropriate attribution for third-party content.
NFR-126. The system SHALL implement mechanisms to prevent copyright
infringement by users.
